Welcome to the FOSSI Github Index

This index contains information about open source silicon repos on Github. This index has been created from lists of Github users and Github repositories. Please find instructions how to add repositories in the README file.

Searching and Sorting

The index can be searched by entering a search query in the search box on top of the table. The search is not case sensitive. Multiple strings can be used. The first three columns are searchable. For instance to search for UART's in VHDL enter:

uart vhdl

To sort click on the header of the column you would like to search. Click again to change the sort direction.
